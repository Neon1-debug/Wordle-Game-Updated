Wordle: простая игра для тренировки ума
Привет! Представляю вам мою первую разработку — увлекательную игру Wordle. Это не просто игра, а настоящий вызов для вашего словарного запаса и логического мышления!
Суть игры предельно проста: вам нужно угадать загаданное слово за 6 попыток. После каждого предположения система подсказывает, какие буквы стоят на своих местах, а какие — нет.

[√]— буква на своём месте
[*]— буква есть в слове, но в другой позиции
[×]— буквы нет в загаданном слове

Особенности игры:
Минималистичный интерфейс без лишних элементов
Простой и понятный геймплей
Никаких регистраций и аккаунтов
Полная автономность — работает без интернета
Подходит для всех возрастов
Как играть:
Введите любое пятибуквенное слово
Смотрите подсказки
Используйте логику для следующих попыток
Побеждайте!
Технически игра реализована на базовом уровне — чистый код без излишеств. Никаких сложных алгоритмов и библиотек — только самое необходимое для работы.
Попробуйте свои силы прямо сейчас! Кто знает, может именно вы станете мастером Wordle и угадаете слово с первой попытки.
P.S. Это мой первый опыт в разработке игр, так что критика приветствуется!
